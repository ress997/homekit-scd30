# HomeKit CO2 sensor SCD30

An Apple HomeKit accessory for the [Adafruit SCD30 CO2 sensor](https://www.adafruit.com/product/4867).

## Software

* Build: `go build homekit-scd30.go`
* Run: `go run homekit-scd30.go`
